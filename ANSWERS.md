
- [ ] What are PropTypes used for? Please describe why it's important to type check our data in JavaScript.

PropTypes are a built in debugging feature in React. It allows us to ensure that the data we are passing around as props matches the data TYPE required for our app to function successfully. It is important to type check our data because JavaScript is a non type specific language, meaning you can pretty much do whatever with whatever.
